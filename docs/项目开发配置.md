# .vscode 目录

用于配置只适用于当前工作区的 vscode 开发环境, 参考[工作区设置](https://code.visualstudio.com/docs/editor/settings#_workspace-settings)

## settings.json

项目专属的编辑器配置（覆盖用户全局设置）

```json
{
  // 启用文件树嵌套
  "explorer.fileNesting.enabled": true,
  // 文件树嵌套规则
  "explorer.fileNesting.patterns": {
    // 匹配 tsconfig.json 文件, 嵌套 tsconfig.*.json, env.d.ts 文件
    "tsconfig.json": "tsconfig.*.json, env.d.ts",
    // 匹配 vite.config.* 文件, 嵌套 jsconfig*, vitest.config.*, cypress.config.*, playwright.config.*, uno.config.* 文件
    "vite.config.*": "jsconfig*, vitest.config.*, cypress.config.*, playwright.config.*, uno.config.*",
    // 匹配 package.json 文件, 嵌套 package-lock.json, pnpm*, .yarnrc*, yarn*, .eslint*, eslint*, .prettier*, prettier*, .editorconfig 文件
    "package.json": "package-lock.json, pnpm*, .yarnrc*, yarn*, .eslint*, eslint*, .prettier*, prettier*, .editorconfig"
  },
  // 保存时的操作
  "editor.codeActionsOnSave": {
    // 保存时自动修复, 安装 ESLint 拓展才能生效
    "source.fixAll": "explicit"
  },
  // 保存时自动格式化
  "editor.formatOnSave": true,
  // 默认格式化工具
  "editor.defaultFormatter": "esbenp.prettier-vscode"
}
```

## extensions.json

项目专属的拓展配置

```json
{
  // 推荐拓展
  "recommendations": [
    // Vue 官方拓展, 高亮、提示、补全 Vue 代码
    "Vue.volar",
    // Vitest 官方拓展, 测试用例管理
    "vitest.explorer",
    // ESLint 拓展, 代码检查
    "dbaeumer.vscode-eslint",
    // EditorConfig 拓展, 跨开发环境保持代码风格一致
    "EditorConfig.EditorConfig",
    // Prettier 拓展, 格式化代码
    "esbenp.prettier-vscode",
    // UnoCSS 拓展, 高亮、提示、补全样式
    "antfu.unocss"
  ]
}
```
